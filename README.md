# AgentNet Help Centre UX Prototype

Static, GitHub Pages-ready prototype for the reimagined PropertyGuru AgentNet Help Centre.

## Pages included

- Remade Help Centre landing page
- How To Purchase Credits Online?
- Renewing Your Agent Package
- Ad Credits Top Up Limit
- Listing Performance Insights

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html` and `.nojekyll` to the repository root.
3. Go to Settings → Pages.
4. Set Source to Deploy from a branch.
5. Choose `main` and `/root`.
6. Save. GitHub will publish the live URL after the Pages build completes.

No build step is required.
